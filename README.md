# Tracker OS: Gamified Habit Dashboard 🐧

Transform your daily routines into a video game. This is a fully interactive, browser-based habit tracker that visualizes your daily progress, calculates statistics in real-time, and auto-saves your data locally. No servers, no databases, no subscriptions.

Built by The Penguin Coder 💻
🌐 **Live Website:** (https://keen-pavlova-88763e.netlify.app/) | (https://penguin-saezo.github.io/Habit-traker)

## ✨ Features

* **🎮 Gamified Analytics:** Live-updating charts that make completing tasks feel like leveling up.
* **📊 Beautiful Visualizations:** Integrated with Chart.js to display Daily Progress (Bar), Weekly Progress (Bar), Overall Stats (Donut), and Wellness Trends (Line).
* **💾 Zero-Setup Auto-Save:** Uses browser localStorage. Your progress saves automatically the second you click a checkbox. Close the tab, come back later, and your data is exactly where you left it.
* **📅 Dynamic Calendar:** Select any Year and Month from the dropdown. The grid automatically adjusts to the correct number of days (28, 30, or 31) and aligns the days of the week perfectly.
* **📝 Fully Customizable:** Click directly on any habit name (e.g., "Wake up at 05:00") to type and rename it. Add new habits on the fly.
* **🧘 Wellness Tracking:** Dedicated tracking rows for Mood and Hours of Sleep, mapped to a custom health chart.

## 🛠️ Tech Stack

This project is lightweight and runs entirely on the frontend.

* HTML5 (Structure & Grid Layout)
* CSS3 (Custom UI, Flexbox/Grid architecture, sleek color palette)
* Vanilla JavaScript (Logic, Math, DOM manipulation, LocalStorage)
* Chart.js (Data visualization)

## 🚀 How to Run

Because this is a standalone web application, installation takes literally one second.

1. Download or clone the HabitTracker.html file to your computer.
2. Double-click the file to open it in your preferred web browser (Google Chrome, Edge, Safari, or Brave).
3. Start checking off your daily quests!

## 🖌️ Customization

If you want to dive into the code and customize the default loadout, open HabitTracker.html in any text editor (like VS Code or Notepad).

* **Change Default Habits:** Search for globalHabitNames in the `<script>` tag and edit the array to fit your ideal daily routine.
* **Change Colors:** Scroll to the top `<style>` section. All colors are controlled via CSS variables in the `:root` block for easy theming.
